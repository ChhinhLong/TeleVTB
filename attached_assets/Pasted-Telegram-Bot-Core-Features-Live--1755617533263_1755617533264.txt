Telegram Bot
មុខងារសំខាន់ៗ (Core Features)
 * មុខងារស្វ័យប្រវត្តិ: នៅពេលមានវីដេអូថ្មី ឬ Live ថ្មី ក្នុង YouTube Channel ដែលបានកំណត់ Bot នឹងចែករំលែកតំណទៅក្នុង Group ដោយស្វ័យប្រវត្តិ។
 * ការគ្រប់គ្រង Command: អ្នកប្រើប្រាស់អាចប្រើ Command ដើម្បីគ្រប់គ្រង Bot ដូចជាការបន្ថែម ឬលុប YouTube Channel ។
 * ការគ្រប់គ្រងសិទ្ធិ: Bot អាចគ្រប់គ្រងសិទ្ធិរបស់ខ្លួនឯងនៅក្នុង Group ដើម្បីធានាថាវាដំណើរការបានត្រឹមត្រូវ។
Command និងមុខងារ (Commands & Functions)
1. /AddYTC <URL>
 * មុខងារ: បន្ថែម YouTube Channel សម្រាប់ឱ្យ Bot តាមដាន។
 * របៀបប្រើ:
   * អ្នកគ្រប់គ្រង Group វាយ Command /AddYTC រួចតាមដោយតំណ URL របស់ YouTube Channel ដូចជា /AddYTC https://www.youtube.com/@Channel_Name។
   * Bot នឹងឆ្លើយតបថា "YouTube Channel បានបន្ថែមដោយជោគជ័យ!"។
 * លក្ខខណ្ឌ: មានតែអ្នកគ្រប់គ្រង Group (Admin) ប៉ុណ្ណោះដែលអាចប្រើ Command នេះបាន។
2. /RemoveYTC <URL>
 * មុខងារ: លុប YouTube Channel ចេញពីបញ្ជីដែល Bot កំពុងតាមដាន។
 * របៀបប្រើ:
   * អ្នកគ្រប់គ្រងវាយ Command /RemoveYTC រួចតាមដោយតំណ URL របស់ Channel ដែលចង់លុប ដូចជា /RemoveYTC https://www.youtube.com/@Channel_Name។
   * Bot នឹងឆ្លើយតបថា "YouTube Channel ត្រូវបានលុបចេញហើយ!"។
 * លក្ខខណ្ឌ: មានតែអ្នកគ្រប់គ្រង Group (Admin) ប៉ុណ្ណោះដែលអាចប្រើ Command នេះបាន។
3. /Group
 * មុខងារ: ផ្តល់សិទ្ធិ (permission) ឱ្យ Bot អាចចែករំលែកតំណបាននៅក្នុង Group។
 * របៀបប្រើ:
   * Command នេះត្រូវប្រើដោយ Bot ខ្លួនឯង ឬដោយអ្នកបង្កើត Bot។ នៅពេល Bot ត្រូវបានបន្ថែមចូលក្នុង Group អ្នកគ្រប់គ្រង Group ត្រូវផ្តល់សិទ្ធិឱ្យវាជា Admin។
   * អ្នកគ្រប់គ្រង Group អាចប្រើ Command /Group ដើម្បីបញ្ជាក់ថា Bot ត្រូវបានផ្តល់សិទ្ធិជា Admin នៅក្នុង Group។
 * ចំណាំ: Command នេះអាចបង្កើតឡើងដើម្បីជួយ Bot ពិនិត្យមើលសិទ្ធិរបស់វា ដោយមិនចាំបាច់ឱ្យអ្នកប្រើប្រាស់វាយដោយផ្ទាល់។
4. /ListYTC (មុខងារបន្ថែម)
 * មុខងារ: បង្ហាញបញ្ជី YouTube Channel ទាំងអស់ដែល Bot កំពុងតាមដាននៅក្នុង Group នោះ។
 * របៀបប្រើ:
   * អ្នកគ្រប់គ្រង Group វាយ Command /ListYTC។
   * Bot នឹងឆ្លើយតបដោយបង្ហាញបញ្ជីឈ្មោះ Channel និងតំណ URL ទាំងអស់។
5. /Help (មុខងារបន្ថែម)
 * មុខងារ: បង្ហាញព័ត៌មានណែនាំអំពីរបៀបប្រើប្រាស់ Bot និង Command ទាំងអស់។
 * របៀបប្រើ:
   * អ្នកប្រើប្រាស់វាយ Command /Help។
   * Bot នឹងឆ្លើយតបដោយបង្ហាញបញ្ជី Command និងការណែនាំខ្លីៗ។
ការរៀបចំសារ (Message Formatting)
 * នៅពេល Bot ផុសវីដេអូថ្មី វានឹងបង្កើតសារដែលទាក់ទាញ និងងាយស្រួលមើល។
 * រចនាសម្ព័ន្ធសារ:
   * ចំណងជើងវីដេអូ: **New Video:** [Video Title]
   * ឈ្មោះ Channel: **From:** [Channel Name]
   * តំណភ្ជាប់: [Watch Here](https://www.youtube.com/watch?v=Video_ID)
   * រូបភាព Thumbnail: Bot អាចបង្ហាញរូបភាពតូច (thumbnail) របស់វីដេអូដើម្បីទាក់ទាញការចាប់អារម្មណ៍។
មុខងារសំខាន់ៗផ្សេងទៀត
 * សារស្វាគមន៍: នៅពេល Bot ត្រូវបានបន្ថែមទៅក្នុង Group ថ្មី វានឹងផ្ញើសារស្វាគមន៍ និងណែនាំ Command /Help។
 * ការពិនិត្យតាមដាន (Polling): Bot គួរតែពិនិត្យមើល Channel YouTube សម្រាប់វីដេអូថ្មីរៀងរាល់ 10-15 នាទីម្តង ដើម្បីធានាថាវាមិនរំខានដល់ API របស់ YouTube ពេក។
